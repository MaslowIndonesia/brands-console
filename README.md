# Brands Revenue Loop — Console

Live 24/7 fleet view over every brand revenue loop (ODP live; RoyalMate, GentleHaus next).
Static site: `index.html` renders `data.json`, generated by `publish.py` ($0, stdlib) from
the brands-revenue-loop registry + member-loop ledgers. Honest by construction — every value
traces to a ledger row; missing data renders as "no data yet", never fabricated.

Refresh: each member-loop run ends with `python3 publish.py --push`.
